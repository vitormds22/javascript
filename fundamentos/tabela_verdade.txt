OPERADOR AND / E
VERDADEIRO + VERDADEIRO => VERDADEIRO
VERDADEIRO + FALSO => FALSO

//SE A PRIMEIRA CONDIÇÃO FOR FALSA CONSEQUENTEMENTE SERÁ FALSO SEM PRECISAR ANALISAR A SEGUNDA CONDIÇÃO
FALSO + FALSO OU VERDADEIRO => FALSO

//BASTA APENAS UM OPERADOR FALSO PARA QUE SEJA UM RETORNO FALSO
VERDADEIRO + VERDADEIRO + VERDADEIRO + VERDADEIRO + FALSO => FALSO

OPERADOR OR / OU

FALSO + FALSO => FALSO
FALSO + VERDADEIRO => VERDADEIRO

//BASTA UM OPERANDO VERDADEIRO PARA QUE A CONDIÇÃO RETORNE VERDADEIRO
FALSO + FALSO + FALSO + VERDADEIRO => VERDADEIRO

OPERADOR XOR / OU EXCLUSIVO

VERDADEIRO + VERDADEIRO => FALSO
VERDADEIRO + FALSO => VERDADEIRO
FALSO + VERDADEIRO => VERDADEIRO
FALSO + FALSO => FALSO
